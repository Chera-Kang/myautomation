## 구성
- 📂 [Web](./Web)  
  - Selenium 기반, Web 서비스 테스트 스크립트 (제약영업관리 서비스 "팜플")  
- 📂 [App](./App)  
  - Appium 기반, *Sample* App 테스트 스크립트 (Wikipedia)

### 작성중
