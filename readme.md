
# 자동화 테스트
Web 및 App 환경에서 수행 중인 자동화 테스트 스크립트를 정리한 저장소입니다.

## 구성
- 📂 [Web](./Web)  
  - Selenium 기반, Web 서비스 테스트 스크립트 (제약영업관리 서비스 "팜플")  
  
- 📂 [App](./App)  
  - Appium 기반, *Sample* App 테스트 스크립트 (Wikipedia)

> 각 폴더에는 테스트 항목별 코드 및 실행 방법이 포함되어 있습니다.
