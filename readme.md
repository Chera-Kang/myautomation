#### 📦 ZIP 파일 구성
- 📁 `screenshots/`  
  - 테스트 실행 중 촬영된 주요 화면 스크린샷 모음입니다.
- 📄 `output.xml`  
  - Robot Framework 실행 결과를 XML 형식으로 저장한 원본 로그 파일입니다.
- 📄 `log.html`  
  - 테스트 상세 실행 과정을 확인할 수 있는 HTML 로그입니다. 각 단계별 상태 및 메시지를 포함합니다.
- 📄 **`report.html`**  
  - **테스트 요약 리포트**로, **전체 테스트 결과**(성공/실패 케이스 등)를 한눈에 파악할 수 있습니다.
